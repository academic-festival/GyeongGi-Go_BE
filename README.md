## 📖 프로젝트 소개

K-컬처의 인기로 한국을 방문하는 외국인 관광객은 많아졌지만, 이들이 **경기도**의 수많은 관광지를 깊이 있게 경험할 방법은 여전히 부족합니다. 기존 여행 앱들은 지도, 길찾기 등 **기술적 정보**에 집중되어 있어, 관광지가 가진 **풍부한 역사적, 문화적 맥락**을 전달하는 데 한계가 있었습니다.

`GyeongGi-Go`는 이러한 **'정보 접근성'**과 **'가이드의 부재'** 문제를 해결하기 위해 탄생한 **외국인 관광객을 위한 경기도 관광 큐레이션 서비스**입니다.

단순히 장소를 나열하는 것을 넘어, 마치 **전문 가이드가 동행**하는 것처럼 각 관광지에 얽힌 흥미로운 이야기와 의미를 전달합니다. 사용자는 **위치 기반 추천**, **AI 챗봇(Gemini API)**을 통한 실시간 질의응답, 그리고 **TTS 오디오 큐레이션**을 통해 여행을 단순한 '방문'이 아닌, 잊을 수 없는 '경험'으로 만들 수 있습니다.

<br>

## 💻 프로젝트 배경
* **정보 전달의 부족** : 현재 외국인 대상 관광 큐레이션 애플리케이션은 주로 지도 기반의 동선 안내, 길 찾기, 단순 일정/장소 정보 제공 등 기능적인 측면에만 집중되어 있습니다.

* **역사적 정보 접근성 문제** : 외국인 관광객은 K-컬처를 통해 한국에 대한 흥미를 느끼고 방문했지만, 정작 관광지에서는 깊은 배경지식을 얻기 어렵습니다. 나무위키 등과 같은 위키형 사이트에서 제공하는 정보는 다수가 편집할 수 있기에 신뢰도가 떨어지고 지역 공식 홈페이지를 통해 정보를 얻기에는 지역별로 정보가 분산 돼있고 제대로 번역된 정보를 얻기 어렵다는 문제가 있습니다.

* **가이드의 부재 및 아쉬움** : 국내 관광객이 해외 패키지여행 시 가이드의 큐레이션을 통해 여행을 더 오래, 감동적으로 기억하는 것처럼, 외국인 개별 관광객에게도 이와 같은 큐레이션이 필요하지만, 기존 애플리케이션은 이러한 가이드 경험이 사실상 부재합니다. 이로 인해 관광지의 매력이 충분히 전달되지 못하고 단순한 SNS용 여행으로 끝나 기억 지속성이 떨어질 수 있습니다.

<br>

## ⭐ 주요 기능
* **[위치기반 관광지 추천]**: 사용자의 위치를 기반으로 주변 관광지 추천 기능 제공
* **[AI Chatbot 기능]**: 제미나이 API를 연동하여 관광지에 대해 질문할 수 있는 chatbot 기능 제공
* **[TTS기반 오디오 큐레이션]**: 오디오로 큐레이션을 들을 수 있는 TTS 기능 제공
